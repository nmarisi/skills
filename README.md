# Agent Skills

A collection of agent skills for planning, review, and development workflows.

## Planning & Design

- **analyse-plan** — Analyse a written plan or spec for logical consistency, decomposition, and completeness. Reports findings plainly and asks up to five targeted questions only where the plan is genuinely unresolvable. A focused alternative to relentless grilling.

  ```
  npx skills@latest add nmarisi/skills/analyse-plan
  ```
